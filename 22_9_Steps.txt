### Vault HCP (Manual Steps)
Step 1.-Enable JWT Auth method
     # vault auth enable JWT
  verify
     # vault auth list
Step 2- Write vault configuration
    # vault write auth/jwt/config \
      oidc_discovery_url="https://app.terraform.io" \
      bound_issuer="https://app.terraform.io"
  verify
    # vault read auth/jwt/config
Step 3- Create Vault Policy
     # create admin-policy.hcl file
     # vault policy write tfc-policy tfc-policy.hcl
  verify
     # vault policy list
     # vault policy read admin-policy
Step 4- Create JWT Auth Role
     create json file (vault-jwt-auth-role.json) ## change organization & project name
     # vault write auth/jwt/role/ vault-jwt-auth-role @vault-jwt-auth-role.json
  verify
     # vault list auth/jwt/role
     # vault read auth/jwt/role/vault-jwt-auth-role
